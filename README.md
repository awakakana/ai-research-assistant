# AI研究助手（ai-research-assistant）

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20110357.svg)](https://doi.org/10.5281/zenodo.20110357)

> **让有限变得有限，让无限变得不远**
> *Let the finite be finite, let the infinite be not far.*

一套以Prompt为核心的AI辅助研究工作流——让研究者专注于思考，让AI处理流程。


## 杂谈

宇宙是无限的，但飞船是有限的。

个人的知识、能力、时间、注意力——都是有限的。

不追求“全能AI”，不幻想“包罗万象”。

做的是：

- **让有限变得有限**：承认边界，善用边界，在有限内做到最优
- **让无限变得不远**：用有限的方法，逼近无限的可能

**这套方法论，就是有限飞船的导航图。**


## 项目定位：方法论落地实践

**这不是一个“AI工具包”，而是一套“人机协作方法论的落地实例”。**

本项目是一套结构化的角色Prompt工具集，将AI转化为可调用的“研究专家”——设计审查官、缺漏分析师、论文结构师等。用户无需学习提示词工程，只需**复制Prompt到AI对话窗口**，即可获得结构化输出。

**零门槛**：无需安装、注册或付费，任何AI对话工具都能用。

**18+角色**：覆盖从模糊想法到答辩演练的全流程。

**可扩展**：通过“角色创建引导师”创建自己的角色。

## 快速开始

### 第一步：选择起始阶段

| 你的情况 | 起始阶段 | 操作 |
|---------|---------|------|
| 第一次使用，不知道从哪开始 | **阶段0** | 使用[全局引导师](阶段0_全局引导/全局引导师.md) |
| 有模糊想法，不知道怎么设计实验 | **阶段1** | 使用[点子孵化器](阶段1_创意孵化/点子孵化器.md) |
| 有实验方案，需要审查优化 | **阶段2** | 使用[设计审查官](阶段2_研究执行/设计审查官.md) |
| 有数据，准备写论文 | **阶段2** | 使用[论文结构师](阶段2_研究执行/论文结构师.md) |
| 准备答辩/组会 | **阶段3** | 使用[组会模拟员](阶段3_学术表达/组会模拟员.md) |
| 有A领域经验，想转到B领域 | **阶段x** | 使用[跨域咨询师](阶段x_跨域接入/跨域咨询师.md) |
| **想创建自己的角色** | **角色设计器** | 使用[角色创建引导师](角色设计器/角色创建引导师.md) |

### 第二步：复制Prompt

点击上方角色链接，复制Prompt内容。

### 第三步：粘贴到AI对话

粘贴到任意AI对话工具（推荐DeepSeek或Qwen，免费），AI会按格式输出分析结果。


## 创建自己的角色

如果你需要的角色不在清单中，可以使用**角色创建引导师**：

1. 复制[角色创建引导师](角色设计器/角色创建引导师.md)的Prompt
2. 粘贴到AI对话，按引导描述你需要的角色
3. AI会生成完整的角色Prompt
4. （可选）使用[空白角色模板](角色设计器/空白角色模板.md)自定义

> 💡 这是“元工具”的雏形——用AI创建AI角色。未来将扩展为完整的元工具集（角色审查官、工作流工厂等）。


## 多语言支持

首次使用任意角色时，AI会询问你的语言偏好。当前支持：中文、English。
