# CHANGELOG

## [0.0.1] - 2026-05-10

### 新增
- 初始最小发行版本发布

### 阶段0：全局引导
- 全局引导师 - 诊断用户状态，推荐起始阶段和角色

### 阶段1：创意孵化
- 点子孵化器 - 将模糊想法转化为初步研究框架
- 可行性分析师 - 评估技术难度、时间、经费
- 创新性评估师 - 判断新意，指导文献检索
- 路径规划师 - 输出从想法到实验的路线图

### 阶段2：研究执行
- 阶段2引导师 - 帮助用户在阶段2中选择角色
- 设计审查官 - 审查对照组、样本量、统计方法
- 缺漏分析师 - 识别缺漏、风险、优先级
- 文献助手 - 提取论文核心信息
- 数据化验收员 - 检查数据完整性、异常值
- 论文结构师 - 生成论文大纲、图表建议

### 阶段3：学术表达
- 阶段3引导师 - 帮助用户在阶段3中选择角色
- 组会模拟员 - 多位专家模拟提问

### 阶段x：跨域接入
- 阶段x引导师 - 帮助用户在阶段x中选择角色
- 跨域咨询师 - 能力迁移分析、学习路线图
- 领域知识速成师 - 核心概念、经典文献、资源清单
- 迁移设计顾问 - 实验设计改编
- 文献迁移工具 - 文献范式迁移

### 角色设计器
- 角色设计引导师 - 通过对话设计新角色
- 空白角色模板 - 标准化角色Prompt模板
